Инициализация 3D-глобуса
⚠️
Требуется локальный сервер
Файл открыт через file:// протокол. Cesium использует Web Workers, которые браузеры блокируют при открытии файлов напрямую. Для работы 3D-глобуса нужен локальный HTTP-сервер.
Способ 1 — Просто (рекомендуется):
  1. Скачайте start.bat и flat-earth-3d.html в одну папку
  2. Двойной клик на start.bat — откроется браузер с картой
Способ 2 — Вручную через Python:
  1. Откройте командную строку в папке с файлом
  2. Запустите: python -m http.server 8765
  3. Откройте в браузере: http://localhost:8765/flat-earth-3d.html
Способ 3 — VS Code:
  1. Установите расширение Live Server
  2. Правый клик на flat-earth-3d.html → Open with Live Server
